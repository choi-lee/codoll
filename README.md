# Hamzzi

macOS 데스크톱 위에 살아 움직이는 햄스터(햄타로 스타일) 펫 앱입니다. 물 마시기, 약 먹기 등 일정을 알림으로 받을 수 있습니다.

## 설치 및 실행

### 앱 실행 (빌드된 .app)

`dist/Hamzzi.app`을 더블클릭하면 바로 실행됩니다. Python 설치 불필요.

### 개발 환경에서 실행

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
python run.py
```

### 빌드

```bash
pip install pyinstaller
pyinstaller Hamzzi.spec --noconfirm

python3 -m PyInstaller Hamzzi.spec --noconfirm
PyInstaller Hamzzi.spec --noconfirm
```

빌드 결과물은 `dist/Hamzzi.app`에 생성됩니다.

## 기능

### 햄스터

- 화면 위를 돌아다니며 다양한 상태로 전환 (걷기, 뛰기, 앉기, 자기, 기쁨)
- 드래그하여 위치 이동 가능
- 클릭하면 기뻐함
- 더블클릭하면 애교를 부림 (반짝이는 큰 눈 + 귀여운 메시지)

### 상호작용 (우클릭 메뉴)

- **쓰다듬기** — 머리 위에 손(🫳)이 나타나 좌우로 쓰다듬는 모션 + 하트 파티클
- **먹이주기** — 해바라기씨를 들고 냠냠 먹는 애니메이션 + 볼이 부풀어오름
- **응원받기** — 별 파티클 + 응원 메시지 말풍선

### 알림 스케줄

- 카테고리: 💧 물 / 💊 약 / 📝 할 일 / ✏️ 커스텀
- 반복 설정: 1회, 매일, 평일, 매주(요일 선택), 간격(N분마다)
- 요일 설정: 설정창에서 "목", "월수금" 등 한글로 입력
- 알림 시 햄스터가 화면 중앙으로 이동하며 말풍선 표시
- 말풍선 클릭으로 완료 처리

### 설정 (우클릭 > 설정...)

- **반복 스케줄**: 알림 추가/수정/삭제, 시간/요일/반복 설정
- **날짜 일정**: 텍스트로 날짜별 일정 입력
- **햄스터**: 이동 모드 (하단 고정 / 전체 화면), 이동 속도 조절
- **일반**: 알림 소리 켜기/끄기

### 종료

- 우클릭 > **Pet 종료**

## 공유 방법

1. `dist/Hamzzi.app`을 zip으로 압축
2. 상대방에게 전달
3. 받은 사람은 압축 해제 후 더블클릭으로 실행
4. "확인되지 않은 개발자" 경고 시: 우클릭 > 열기, 또는 시스템 설정 > 개인 정보 보호 및 보안에서 허용

> Python이나 별도 라이브러리 설치 없이 실행 가능합니다. 단, Apple Silicon(arm64) Mac 전용입니다.

## 요구사항

- macOS (Apple Silicon)
- 개발 시: Python 3.12+
