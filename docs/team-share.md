# Puppy - macOS 데스크탑 펫 + 스케줄 리마인더

화면 위에 떠다니는 햄스터가 스케줄에 맞춰 알림을 보내주는 macOS 데스크탑 앱입니다.

---

## 기술 스택

| 구분 | 내용 |
|------|------|
| 언어 | Python 3.12+ |
| UI 프레임워크 | PyObjC (AppKit / Quartz) — macOS 네이티브 |
| 렌더링 | CoreGraphics 직접 렌더링 (32x32 픽셀아트, 2x 스케일) |
| 데이터 저장 | JSON (`~/Library/Application Support/Puppy/`) |
| 빌드 | setuptools, `pip install -e .` 후 `puppy` 명령으로 실행 |

외부 GUI 라이브러리(Electron, Qt 등) 없이 PyObjC만으로 구현했습니다.

---

## 구현된 기능

### 1. 데스크탑 펫 (햄스터)

- 햄토리 스타일 32x32 픽셀아트 캐릭터
- **7가지 상태 애니메이션**: IDLE, WALKING, RUNNING, SITTING, SLEEPING, ALERT, HAPPY
- 상태별 눈/귀/다리/꼬리 변화, 바운스, ZZZ 표시 등
- 30fps 렌더링 루프
- 좌우 반전 자동 전환

### 2. 이동 & 인터랙션

- 화면 하단 또는 전체 화면 자유 이동 (설정 가능)
- 이동 속도 0.3x ~ 3.0x 조절
- 클릭하면 기뻐함 (HAPPY 상태)
- 드래그로 위치 이동
- 우클릭 → 컨텍스트 메뉴 (설정, 종료)
- 캐릭터/말풍선 외 영역은 마우스 이벤트 통과 (데스크탑 조작 방해 안 함)

### 3. 스케줄 알림

- **카테고리**: 물 마시기, 약 먹기, 할 일, 커스텀
- **반복 유형**: 한 번, 매일, 평일, 매주, 커스텀 요일, N분 간격
- 알림 시 캐릭터가 화면 중앙으로 이동 + 말풍선 표시
- 말풍선 클릭 → 완료 처리 + HAPPY 상태
- 알림 사운드 on/off

### 4. 날짜 일정 (평문 입력)

별도 탭에서 텍스트 에디터로 날짜별 일정을 관리할 수 있습니다:

```
#2026-03-30
재택
14:30 팀 회의

#2026-03-31
08:00 출근
12 점심식사
```

- `#날짜` 헤더 아래 일정 나열
- `HH:MM 제목`, `H 제목`, 또는 제목만 (기본 09:00)
- 날짜 형식: `YYYY-MM-DD`, `MM-DD`, `M/D`, `MMDD` 지원

### 5. 설정 창 (4탭)

| 탭 | 내용 |
|----|------|
| 반복 스케줄 | 테이블에서 인라인 편집 (체크박스, 드롭다운, 텍스트 직접 수정) + 저장 |
| 날짜 일정 | 평문 텍스트 에디터로 날짜별 일정 입력 |
| 햄스터 | 활동 범위(하단/전체), 이동 속도 슬라이더 |
| 일반 | 알림 사운드 토글 |

## 아키텍처

```
puppy/
├── app.py                 # 메인 앱, NSWindow, 렌더링 루프, 마우스 처리
├── renderer.py            # 32x32 픽셀아트 렌더링 (CoreGraphics)
├── animator.py            # 위치/상태 업데이트, 이동 로직
├── state.py               # 상태 머신 (7 상태, 전이 확률, 지속 시간)
├── bubble.py              # 말풍선 UI (페이드 인/아웃)
├── schedule.py            # 스케줄 데이터 모델, JSON 저장/로드
├── schedule_manager.py    # 10초 간격 스케줄 체크, 알림 트리거
└── settings_window.py     # 설정 창 UI (4탭, 인라인 테이블, 텍스트 에디터)
```

**동작 흐름**: `main()` → 투명 풀스크린 윈도우 생성 → 30fps 타이머로 매 프레임 `animator.update()` + `renderer.draw_puppy()` → 10초마다 `schedule_manager`가 알림 체크

---

## 실행 방법

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
puppy                # 또는 python3 run.py
```
